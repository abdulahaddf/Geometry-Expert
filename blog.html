What is event bubble?

In the Document Object Model (DOM), an event propagates or "bubbles" up through its parent elements to the outermost or top-level element using a mechanism called event bubble.

What is a callback function and why will you use it?

A callback function is a function that is passed as an argument to another function and is executed when the other function has completed its task. The main advantage of using a callback function is that it allows the calling function to continue executing and performing other tasks.
